# Projeto-CRUD-com-Java
Um projeto simples para a implementação de um CRUD com tabelas no Java
## ProjetoCRUD
O arquivo com o código fonte Java contendo o projeto
## Classe UML
o arquivo PNG contendo o diagrama UML do projeto
## Relatorio
o relatório de uma página sobre o projeto
## Como executar o projeto:
O Projeto foi desenvolvido na versão 17.0.17 do JDK, se utilizando da interface gráfica Swing, junta das bibliotecas ArrayList e JOptionPane. O Projeto foi efetuado na IDE NetBeans, e precisa ser aberto em uma IDE para ser rodado
